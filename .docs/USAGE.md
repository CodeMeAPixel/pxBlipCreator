## Usage

### Opening the UI

Use the command:

```
/blipcreator
```

You must have the `command.blipcreator` or `pxblip.access` permission.

### Blips Tab

- **List**: View all blips (filterable by name)
- **Create**: Click the create button and fill in a name
- **Edit**: Click a blip row to edit properties (general, colors, groups, items, sprites, effects)
- **Delete**: Delete a blip you created (icon in the list)
- **Teleport**: Quick-jump to a blip's location (icon in the list)

### Settings Tab

When editing a blip:

- **General**: Name and coordinates
- **Colors**: Blip color and appearance
- **Groups**: Job/group visibility (only shown to players in these groups)
- **Items**: Item requirement (not yet enforced in-game)
- **Sprites**: Blip sprite ID and custom image
- **Effects**: Flash, outline, scale, alpha, short-range visibility

### Import & Export

From the blips list header:

- **Export**: Download a JSON file of all blips
- **Import**: Upload a previously exported JSON file to restore or merge blips
